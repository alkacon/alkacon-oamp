1. Module functions

With the newsletter module, it is possible to send newsletter emails
created as structured contents to subscribers of different mailing lists.
The mailing lists and subscribers can be easily managed inside OpenCms.
A configurable frontend page allows the subscription or unsubscription to/from
mailing lists for website visitors.
