1. Module functions

The calendar module adds functionality to an existing OpenCms installation 
to display calendar based lists of events and other time based information.

It features JSP side elements to be included in a template showing a monthly
overview or the most recent events. Also daily, weekly, monthly and yearly
overviews can be shown.

Additionally, it is possible to define serial dates appearing more than once.
